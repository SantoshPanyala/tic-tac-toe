🎮 Tic-Tac-Toe (React)

A simple and fun Tic-Tac-Toe game built with React.js! Play, win, and reset anytime!

🔗 Live Demo: Play Here
✨ Features

✅ Turn-based X & O play
✅ Win detection 🎉
✅ Reset button 🔄
✅ Styled UI 🎨
⚡ Run Locally

git clone https://github.com/SantoshPanyala/tic-tac-toe.git
cd tic-tac-toe
npm install
npm start

Runs on http://localhost:3000/ 🚀
🚀 Deploy to GitHub Pages

npm run deploy
